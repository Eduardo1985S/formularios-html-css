📚 Teoria sobre Formulários HTML

O que são Formulários?

Formulários HTML são elementos que permitem aos usuários inserir e enviar dados para um servidor. São fundamentais para interação do usuário com páginas web.

Elementos principais:

  • <form> - Container principal do formulário
  • <input> - Campos de entrada de dados
  • <textarea> - Área de texto multilinhas
  • <select> - Lista de seleção (dropdown)
  • <button> - Botões de ação
  • <label> - Rótulos para acessibilidade

Estrutura básica:

<form action="/submit" method="POST">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    
    <button type="submit">Enviar</button>
</form>

✅ Validação de Formulários

Validação HTML5 (Nativa)

Dicas de Validação:

  • Use required para campos obrigatórios
  • Use pattern para validação com regex
  • Use min e max para números
  • Use minlength e maxlength para texto
  • Tipos específicos como email e url têm validação automática

Seletores CSS Úteis:

/* Estilizar todos os inputs */
input { border: 1px solid #ccc; }

/* Input com foco */
input:focus { border-color: #007bff; }

/* Input obrigatório */
input:required { border-left: 3px solid orange; }

/* Input válido */
input:valid { border-color: green; }

/* Input inválido */
input:invalid { border-color: red; }

/* Placeholder */
input::placeholder { color: #999; }
